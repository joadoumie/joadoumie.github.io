# joadoumie.github.io

Personal site for Jordi Adoumie — PM @ Bloomberg/BQuant, open-source builder, basketball obsessive.

A monkeytype-inspired portfolio with a playable typing test, an interactive ASCII basketball court, a terminal takeover post-session, and a local leaderboard.

## Stack

- Vite + React + TypeScript
- JetBrains Mono (self-hosted via @fontsource)
- Deployed to GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Static output goes to `dist/`. Deployed via GitHub Actions to `joadoumie.github.io`.
